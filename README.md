# Portfólio Pessoal | Raul Victor

Este repositório contém o código-fonte do meu site de portfólio pessoal. O projeto foi desenvolvido para demonstrar as minhas competências em desenvolvimento web, com foco em *creative coding*, performance e na construção de interfaces visuais interativas e cativantes.

## 🚀 Tecnologias e Ferramentas

O projeto parte de um ecossistema moderno para a Web, trazendo a robustez do React com outras bibliotecas:

* **[React](https://react.dev/) (v19):** Biblioteca principal para a construção da interface.
* **[TypeScript](https://www.typescriptlang.org/):** Tipagem estática para garantir um código mais seguro, previsível e escalável.
* **[Vite](https://vitejs.dev/):** Ferramenta de *build* e servidor de desenvolvimento.
* **[Three.js](https://threejs.org/) & [React Three Fiber](https://r3f.docs.pmnd.rs/):** O núcleo da renderização 3D do projeto, permitindo manipular a GPU e criar visuais imersivos diretamente no browser.
* **GLSL (via `vite-plugin-glsl`):** Integração nativa para a escrita de *shaders* personalizados, necessários para dar vida à efeitos visuais complexos como *Mesh Gradient*.
* **[Leva](https://github.com/pmndrs/leva):** Painel de controle flutuante para refinar e depurar parâmetros em tempo real.

## ⚙️ Rodar localmente

### `git clone https://github.com/Raul772/itsme.git`
Clona o repositório para a máquina local

### `npm install`
Instala as dependências do projeto

### `npm run dev`
Inicia o servidor de desenvolvimento local. Abra `http://localhost:5173` em navegador para visualizar a página.
